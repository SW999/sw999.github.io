# FrontCamp

## Getting Started
Please run this code in command line:

```shell
npm install
```


### Usage for Win
Start Dev and api servers - just run file:

```shell
start.bat
```

### Usage for Mac
Start Dev server:

```shell
npm run dev
```

Start Api server:

```shell
npm start
```


Locall URl will be:

```shell
http://localhost:3000/
```


## Testing

#### 1. Unit Tests

* single run: `npm test`
* live mode (TDD style): `npm run test-watch`
